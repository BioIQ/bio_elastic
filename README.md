# BioElastic

Interface for creating documents in BioIQ's elasticsearch document store

## Installation

Add this line to your application's Gemfile:

    gem 'bio_elastic'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bio_elastic
